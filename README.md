# 🔥 Calorie Tracker

Aplicación web para registrar alimentos y actividades físicas, permitiendo calcular calorías consumidas y quemadas.

---

## 🚀 Tecnologías usadas

- React
- TypeScript
- Vite
- useReducer
- Context API
- CSS

---

## ⚙️ Funcionalidades

- Agregar actividades o comidas
- Seleccionar categoría (comida o ejercicio)
- Editar actividades
- Eliminar actividades
- Calcular calorías consumidas y quemadas
- Mostrar balance total
